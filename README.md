# 🌍 Tempus — Time Rendered With Intention

**Tempus** is a visually engaging and interactive world clock web application that allows users to explore time across different regions of the globe. It combines smooth animations, modern UI design, and real-time clock rendering to create an immersive user experience.

---

## 🚀 Features

* 🕒 **Real-Time Clock Rendering** using Canvas API
* 🌍 **Interactive World Map** — Hover over regions to see local time
* 🌐 **Multiple Cities Display** — Scroll to explore time in major cities
* 🎨 **Modern UI/UX Design** with smooth animations
* 📱 **Responsive Layout** for all screen sizes
* ✨ **Scroll-based Animations** powered by GSAP & ScrollTrigger

---

| Technology               | Purpose                                              |

## ⚙️ How It Works

* The **Canvas API** is used to dynamically render clock visuals.
* **JavaScript** calculates real-time time zones.
* **Hover events** trigger time display for specific regions.
* **GSAP + ScrollTrigger** control smooth animations while scrolling.
* Fully responsive design ensures usability across devices.
